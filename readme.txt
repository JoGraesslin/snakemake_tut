~# Snakemake Tutorial: Python → R Workflow

This tutorial demonstrates a simple **Snakemake pipeline** that:

1. Generates random data using **Python**
2. Plots the data using **R (ggplot2)**

Installation of a mamba env containing snakemake is required to run the scripts (See: installation/install_snakemake.sh)
snakefile contains rules for execution of snakemake
environments are automatically installed by snakemake at .snakemake
